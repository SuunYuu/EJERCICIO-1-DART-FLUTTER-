# 📌 Sistema con API, Base de Datos y Frontend Flutter

Este proyecto está compuesto por tres módulos principales:

- 🔧 Backend (APIs en Node.js)
- 🗄️ Base de datos (MySQL)
- 🎨 Frontend (Flutter)

## ⚙️ Instalación y Ejecución

### 🔧 Backend (APIs)

1. Ingresa a la carpeta del backend:
   ```bash
   cd api_basic
````

2. Instala las dependencias:

   ```bash
   npm install
   ```

3. Ejecuta la API:

   ```bash
   npm start
   ```

La API correrá en:

```
http://localhost:3000
```

---

### 🗄️ Base de Datos

1. Abre phpMyAdmin:

```
http://localhost:8080/phpmyadmin
```

2. Crea una nueva base de datos

3. Importa el archivo `.sql` que se encuentra en la carpeta del proyecto

---

### 🎨 Frontend (Flutter)

1. Ingresa a la carpeta del frontend (donde esté `.dart_tool`):

   ```bash
   cd front
   ```

2. Instala dependencias:

   ```bash
   flutter pub get
   ```

3. Ejecuta la aplicación:

   ```bash
   flutter run
   ```
